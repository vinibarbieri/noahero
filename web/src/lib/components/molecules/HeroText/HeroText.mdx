# HeroText

O componente `HeroText` exibe um texto fixo "As empresas que você ama" seguido por um texto rotativo com efeitos de animação sofisticados. É ideal para seções hero de landing pages onde você deseja destacar diferentes benefícios ou características do seu produto de forma dinâmica e atraente.

## Detalhes de Implementação

O componente `HeroText` combina:

1. Um texto fixo "As empresas que você ama"
2. Um texto rotativo que alterna entre as frases:
   - "melhores ainda"
   - "com muito mais vantagem"
   - "te dando muita economia"
   - "com itens exclusivos"

### Efeitos Visuais Aprimorados

- **Sequência de Animações**: O componente utiliza uma sequência coreografada de animações para criar um efeito visual impactante
- **Múltiplos Efeitos de Transição**: Combinação de efeitos `scale`, `fly` e `fade` para transições suaves e elegantes
- **Fundo Gradiente Animado**: Um gradiente nas cores primárias da aplicação com efeito de pulsação
- **Efeito de Brilho Dinâmico**: Um efeito de blur animado que entra com escala para criar profundidade
- **Efeito Hover Aprimorado**: Transição suave de escala e sombra ao passar o mouse sobre o texto rotativo
- **Animação de Caracteres**: O texto rotativo anima cada caractere individualmente com timing preciso

### Responsividade

O componente é totalmente responsivo e se adapta a diferentes tamanhos de tela:

- **Mobile**: Tamanho de fonte reduzido e espaçamento ajustado
- **Tablet**: Tamanho médio com altura ajustada
- **Desktop**: Tamanho maior para melhor impacto visual
